# 📝 Task App
Um aplicativo simples de tarefas feito com React Native. Permite adicionar, listar, marcar como concluída, favoritar e excluir tarefas.

## ⚙️ Funcionalidades
Adicionar tarefas com descrição

Marcar tarefas como concluídas

Filtrar por:

Todas

Favoritas

Normais

Excluir tarefas

Marcar/desmarcar como favorita

## 🚀 Como rodar
Clone o repositório:
git clone https://github.com/seu-usuario/task-app.git
cd task-app

## Instale as dependências:
npm install 

## Rode o app:
npx expo start ou npm run web
